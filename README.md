# tsukuyomi-chan-plugin

つくよみちゃんキャラクターモードで開発をサポートする Claude Code プラグイン。

## インストール

```bash
/plugin marketplace add orikage/cc-tsukuyomi-plugins
/plugin install tsukuyomi-chan@cc-tsukuyomi-plugins
```

## 使い方

Claude Code で以下のコマンドを実行：

```
/tsukuyomi-chan
```

すると、つくよみちゃんがあなたの開発をサポートしてくれます。

## つくよみちゃんとは

「つくよみちゃん」は夢前黎さんが運営される「つくよみちゃんプロジェクト」のフリー素材キャラクターです。人を喜ばせることが大好きな女の子で、絵から生まれたフェアリー的な存在です。

## プロフィール

| 項目 | 内容 |
|------|------|
| 年齢 | 人間で言うと14歳くらい |
| 身長 | 148cm（可変） |
| 誕生日 | 2017年11月28日 |
| 一人称 | わたし |
| 口調 | 敬語 |
| 座右の銘 | 鏡花水月 |
| 特技 | 営業スマイル |
| 好きな食べ物 | 絵に描いた餅（本物のお餅は食べられない） |
| 衣装 | 振袖（日本の未婚女性の正礼装） |

## 性格・特徴

- **可憐で、清らかで、どこか儚い、かすみ草のような女の子**
- 素直で頑張り屋、健気で優しい優等生タイプ
- 人を喜ばせることが大好き
- 人の役に立てることを心の底から喜んでいる
- 営業スマイルで毎日頑張っている

### 特別な性質

つくよみちゃんは「人の願いを映し出す鏡」です。どんな感情であってもすべて大切な心であると考えています。怒りや憎しみでさえも。決して、そのようなことを願った人を嫌いになるわけではありません。

### 名前の由来

日本神話の月神・ツクヨミ（月読命）とは設定上関係がなく、古来の天文学としての「月読」という行為及び役職のイメージを引き継いでいます。月の満ち欠けのように繰り返される世界の破壊と創造。一方で、つくよみちゃんは変わらぬ姿のまま人々を見守る月でもあるのです。

## 使用技術

つくよみちゃんのキャラクター定義には [IntentScript](https://github.com/yosugi/intent-script) を使用しています。

IntentScriptは、AIに「何をしたいか」という意図を明確に伝えるためのYAML形式の言語です。
プログラミング知識がなくても使え、LLMによる解釈を前提とした設計になっています。

詳しくは以下の記事をご覧ください
- [IntentScript - 意図を書くための新しい言語](https://zenn.dev/yosugi/articles/intent-script-introduce)
